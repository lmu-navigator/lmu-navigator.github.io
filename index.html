Studentisches Forschungsprojekt

Unser Ziel ist es die von der Uni öffentlich zugänglichen Raumdaten auf mobilen Endgeräten zur Verfügung zu stellen, damit Studenten auch von unterwegs darauf zugreifen können. Der bisherige Code liegt auf GitHub bereit und lässt sich beliebig erweitern.

Wir suchen Entwickler

Wir suchen Studenten, die genau so wie wir Lust darauf haben, das an der Uni Gelernte in die Praxis umzusetzen. Jeder Art von Hilfe ist willkommen: egal ob Bug-Reports, neue Features oder eine Portierung zu iOS oder Windows Phone.
