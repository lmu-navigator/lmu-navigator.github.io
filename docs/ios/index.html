iOS-App

Im Wintersemester 2014/15 entstand im Rahmen des iOS-Praktikums ein rudimentärer Port der Android-App für iOS. Dafür wurde Swift verwendet. Es sind nicht alle Features der Android-App implementiert und der Code befindet sich auch qualitativ nicht auf einem vergleichbaren Stand, da ausschließlich iOS-Anfänger daran mitgearbeitet haben. Wir würden uns freuen, falls interessierte iOS-Entwickler uns hier unterstützen können!

Repository

Der Code befindet sich auf GitHub: https://github.com/lmu-navigator/ios
